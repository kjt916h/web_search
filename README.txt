# 内容物

1. setup.sh
   概要: 演習室内でpython3.6とpipを扱うためのセットアッププログラム
   実行: sh setup.sh
   ### 注意 ###
       このプログラムは、bash専用です。
       また、.bashrc を上書きするコマンドが含まれています。
       .bashrcを改造している人は、スクリプトを読んで自前で設定するか、生成されるバックアップを元に復元する等の対応をとってください。

2. search_job_json.py
  概要: WebAPIサンプルプログラム(詳しくは、講義スライド参照)

3. .bashrc と .pip.conf
  隠れファイル。setup.sh内で使用
